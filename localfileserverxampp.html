SETTING UP A LOCAL FILE SERVER

  1. Download and install XAMPP here.
  2. Go into the configurations, and choose the text editor of your choice
  3. In the XAMPP folder, locate the htdocs folder
    • this will be the repository for your web pages
    • make a subfolder to hold all the pages for the site you're building
  4. Create a file using the .html tag in VS Code, Notepad, et c.
  5. You should now be able to access your site locally by typing localhost/"folder name" into your browser.
