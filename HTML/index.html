Fun With Tables

Use the table tag to create a table, and the tr tag to create rows.
The th tag is used to create labels for the columns in the first row of the table.
Data can be added to each cell in a row by using the td (table data) tag.

Add a border of various size to the table by adding a border attribute in the opening tag: table border="size(number)".
Add internal spacing to the cells, by adding the cellpadding="(number)" attribute to the tag.
The cellspacing="(size)" attribute adds spacing between the cells.
You can use the colspan="" and rowspan="" attributes to make headers span however many rows or columns you want.
